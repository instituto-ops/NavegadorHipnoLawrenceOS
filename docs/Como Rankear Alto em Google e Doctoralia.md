Sou psicólogo clínico, especialista em Transtorno do Espectro Autista (TEA) e expert em Hipnose Clínica Ericksoniana. Tenho vasta experiência no tratamento de diversos transtornos psicológicos e emocionais, utilizando abordagens focadas como a psicoterapia breve e estratégica. Com mais de 10 mil horas de prática clínica, realizo atendimento online para todo o Brasil.

Pesquisa e Inovação em TEA

Como pesquisador, dedico-me há mais de 12 anos à construção e validação de instrumentos diagnósticos para TEA em adultos. Sou autor da primeira escala válida para rastreio de TEA em adultos no Brasil (AQ10b, 2018) e, atualmente, realizo a validação de um novo instrumento robusto para colaborar no processo diagnóstico do TEA na rede do SUS.

A Profundidade da Hipnose Ericksoniana

Paralelamente, sou um dos poucos pesquisadores no Brasil a pesquisar cientificamente de forma ética e profunda a hipnose moderna ericksoniana. Minha linha de pesquisa foca em aspectos teóricos avançados, como o "uso simbólico do fenômeno hipnótico" — um campo de estudo com notável singularidade no cenário acadêmico nacional. Este estudo foca no uso da hipnose para produzir mudanças psicológicas profundas e duradouras.

Uma Abordagem Inovadora e Eficaz

A união dessas frentes — clínica, pesquisa, hipnose e autismo — permitiu-me construir uma perspectiva inovadora para entender e manejar o TEA em adultos. Essa abordagem integrada se mostra eficiente, muitas vezes surpreendendo pacientes que já haviam desistido de terapias tradicionais.
Convido você a ler as opiniões públicas de outros pacientes com TEA aqui no meu perfil para conhecer os resultados desse trabalho.

Como Posso Ajudar

Atendo adultos, casais e famílias, ajudando a superar uma diversidade de questões psicológicas, emocionais, de relacionamento e até profissionais. A hipnose clínica é um recurso eficaz para tratar qualquer condição psicológica e emocional, como ansiedade, depressão, transtornos do sono, compulsões, dor crônica, além de promover autoconhecimento. Também utilizo técnicas de Programação Neurolinguística (PNL) para ampliar as possibilidades de mudança.

Meu atendimento é humanizado e acolhedor, sempre respeitando sua história, ritmo e necessidades. Tenho formação avançada em Hipnose Clínica pela Milton H. Erickson Foundation (EUA), referência internacional em hipnose clínica e psicoterapia breve.

#!/usr/bin/env python3
# -*- coding: utf-8 -*-

"""
SCRIPT: doctoralia_check.py
OBJETIVO: Rastrear múltiplos termos de busca na Doctoralia em batch processing
CONTEXTO: NeuroStrategy OS - DOCTORALIA_Radar
DATA: 16/02/2026
AUTOR: Victor Lawrence Bernardes Santana

import json
import random
import time
import requests
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC
from datetime import datetime
import logging

# ============================================================================
# CONFIGURAÇÃO DE LOGGING
# ============================================================================

logging.basicConfig(
    level=logging.INFO,
    format='%(asctime)s - %(levelname)s - %(message)s',
    handlers=[
        logging.FileHandler('doctoralia_batch.log'),
        logging.StreamHandler()
    ]
)
logger = logging.getLogger(__name__)

# ============================================================================
# LISTA DE TERMOS DE PESQUISA (17 TERMOS + EXPANDIDOS)

SEARCH_TERMS = 

    # Queixas relacionais / burnout autista
    'psicólogo online para ansiedade social em adultos',

    # Psicoterapia breve / sintomas gerais online

    'atendimento psicológico online brasil',
]

def inicializar_driver():
    """Inicializa o webdriver do Selenium com opções otimizadas."""
    options = webdriver.ChromeOptions()
    options.add_argument('--start-maximized')
    options.add_argument('--disable-blink-features=AutomationControlled')
    options.add_argument('user-agent=Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36')
    
    driver = webdriver.Chrome(options=options)
    return driver

# ============================================================================
# FUNÇÕES DE BUSCA E EXTRAÇÃO
# ============================================================================

def buscar_termo_doctoralia(driver, termo):
    """
    Executa busca de um termo na Doctoralia e extrai dados relevantes.
    
    Args:
        driver: Instância do Selenium webdriver
        termo: String do termo de busca
    
    Returns:
        dict com dados da busca ou None se falhar
    """
    try:
        url_busca = f"https://www.doctoralia.com.br/search?q={termo.replace(' ', '+')}"
        logger.info(f"Iniciando busca para termo: '{termo}'")
        
        driver.get(url_busca)
        
        # Aguardar carregamento dos resultados
        WebDriverWait(driver, 10).until(
            EC.presence_of_all_elements_located((By.CLASS_NAME, "doctor-card"))
        )
        
        time.sleep(random.uniform(2, 4))
        
        # Extrair dados dos resultados
        resultados = []
        elementos = driver.find_elements(By.CLASS_NAME, "doctor-card")
        
        for idx, elemento in enumerate(elementos[:5], 1):  # Top 5
            try:
                nome = elemento.find_element(By.CLASS_NAME, "doctor-name").text
                especialidade = elemento.find_element(By.CLASS_NAME, "specialty").text
                url_perfil = elemento.find_element(By.TAG_NAME, "a").get_attribute("href")
                avaliacao = elemento.find_element(By.CLASS_NAME, "rating").text if elemento.find_elements(By.CLASS_NAME, "rating") else "N/A"
                
                resultados.append({
                    "posicao": idx,
                    "nome": nome,
                    "especialidade": especialidade,
                    "url_perfil": url_perfil,
                    "avaliacao": avaliacao
                })
            except Exception as e:
                logger.warning(f"Erro ao extrair elemento {idx} para '{termo}': {e}")
                continue
        
        logger.info(f"✅ Termo '{termo}': {len(resultados)} resultados extraídos")
        
        return {
            "termo": termo,
            "status": "sucesso",
            "timestamp": datetime.now().isoformat(),
            "quantidade_resultados": len(resultados),
            "resultados": resultados
        }
        
    except Exception as e:
        logger.error(f"❌ Erro ao buscar termo '{termo}': {str(e)}")
        return {
            "termo": termo,
            "status": "erro",
            "timestamp": datetime.now().isoformat(),
            "erro": str(e),
            "quantidade_resultados": 0,
            "resultados": []
        }

def tirar_screenshot_diagnostico(driver, nome_arquivo):
    """Tira screenshot de diagnóstico para debug."""
    try:
        driver.save_screenshot(f"debug_{nome_arquivo}.png")
        logger.info(f"📸 Screenshot salvo: debug_{nome_arquivo}.png")
    except Exception as e:
        logger.warning(f"Aviso ao tirar screenshot: {e}")

# ============================================================================
# FUNÇÃO PRINCIPAL — BATCH PROCESSING
# ============================================================================

def processar_batch_termos():
    """
    Processa TODOS os termos em SEARCH_TERMS, acumula resultados,
    e retorna JSON para n8n.
    """
    logger.info(f"\n{'='*70}")
    logger.info(f"INICIANDO BATCH PROCESSING — {len(SEARCH_TERMS)} TERMOS")
    logger.info(f"{'='*70}\n")
    
    driver = inicializar_driver()
    resultados_totais = []
    
    try:
        for idx, termo in enumerate(SEARCH_TERMS, 1):
            logger.info(f"\n[{idx}/{len(SEARCH_TERMS)}] Processando: '{termo}'")
            
            try:
                # Buscar e extrair
                resultado = buscar_termo_doctoralia(driver, termo)
                resultados_totais.append(resultado)
                
                # Screenshot a cada 5 termos
                if idx % 5 == 0:
                    tirar_screenshot_diagnostico(driver, f"batch_{idx}")
                
            except Exception as e:
                logger.error(f"Erro crítico ao processar '{termo}': {e}")
                resultados_totais.append({
                    "termo": termo,
                    "status": "erro_crítico",
                    "timestamp": datetime.now().isoformat(),
                    "erro": str(e),
                    "quantidade_resultados": 0,
                    "resultados": []
                })
            
            # ⭐ DELAY HUMANO (OBRIGATÓRIO) ⭐
            delay = random.uniform(5, 12)
            logger.info(f"⏸️  Aguardando {delay:.1f}s antes do próximo termo...")
            time.sleep(delay)
    
    finally:
        driver.quit()
        logger.info("✅ Webdriver encerrado com sucesso")
    
    # ========================================================================
    # SAÍDA FINAL: JSON PARA N8N
    # ========================================================================
    
    resultado_final = {
        "timestamp_inicio": datetime.now().isoformat(),
        "total_termos": len(SEARCH_TERMS),
        "termos_processados": len(resultados_totais),
        "sucessos": sum(1 for r in resultados_totais if r["status"] == "sucesso"),
        "erros": sum(1 for r in resultados_totais if r["status"] in ["erro", "erro_crítico"]),
        "dados": resultados_totais
    }
    
    logger.info(f"\n{'='*70}")
    logger.info(f"BATCH COMPLETO")
    logger.info(f"Sucessos: {resultado_final['sucessos']}/{len(SEARCH_TERMS)}")
    logger.info(f"Erros: {resultado_final['erros']}/{len(SEARCH_TERMS)}")
    logger.info(f"{'='*70}\n")
    
    # ✅ PRINT JSON ÚNICO PARA N8N
    print(json.dumps(resultado_final, ensure_ascii=False, indent=2))
    
    # Salvar também em arquivo para referência
    with open(f"doctoralia_batch_{datetime.now().strftime('%Y%m%d_%H%M%S')}.json", 'w', encoding='utf-8') as f:
        json.dump(resultado_final, f, ensure_ascii=False, indent=2)
    
    return resultado_final

# ============================================================================
# EXECUÇÃO
# ============================================================================

if __name__ == "__main__":
    processar_batch_termos()
