## PARTE 3: CONFIGURAÇÕES DE ORÇAMENTO E LANCES

### SEÇÃO: Budget & Bidding

```
CAMPO 1: Orçamento Diário
────────────────────────────────────
Localização: Campaign Settings → Budget

VALOR RECOMENDADO:
├─ Fase Teste (Sem. 1-2): R$ 50-75/dia
├─ Fase Crescimento (Sem. 3-4): R$ 100-150/dia
├─ Fase Otimização (Mês 2-3): R$ 150-200/dia
└─ Fase Scaling (Mês 4+): R$ 300-500/dia

CÁLCULO:
├─ Meta: 5 consultas/mês
├─ Cliques necessários: 25-35 (com 15-20% conversão)
├─ CPC estimado: R$ 2,50-3,50
├─ Gasto: 30 cliques × R$ 3,00 = R$ 90/dia
└─ Recomendado: R$ 100/dia (com 20% margem)

⚠️ IMPORTANTE:
├─ Este é o MÁXIMO que gastará por dia
├─ Google não gastará mais que isso
├─ Mas pode gastar menos se poucas buscas
└─ Você consegue alterar a qualquer momento

─────────────────────────────────────

CAMPO 2: Método de Lances (Bidding Strategy)
────────────────────────────────────
Localização: Campaign Settings → Bidding

OPÇÃO 1: Maximizar Cliques ⭐ SEMANA 1
┌──────────────────────────────────────┐
│ Google otimiza para CLIQUES          │
│ Sem limite de CPC                    │
│ Bom para: Coletar dados iniciais     │
└──────────────────────────────────────┘

OPÇÃO 2: Maximizar Conversões ⭐ SEMANA 2+ (RECOMENDADO)
┌──────────────────────────────────────┐
│ Google otimiza para CONVERSÕES       │
│ IA tenta atingir máximo de agendam.  │
│ Limite CPA (opcional): R$ 100-150    │
│ Bom para: Iniciantes, eficiência     │
└──────────────────────────────────────┘

OPÇÃO 3: Target CPA (Depois)
┌──────────────────────────────────────┐
│ Você define: "Quero gastar R$ X      │
│ por conversão"                       │
│ Requer: 20+ conversões históricas    │
│ Bom para: Campanha madura            │
└──────────────────────────────────────┘

OPÇÃO 4: Target ROAS (Avançado)
┌──────────────────────────────────────┐
│ "Quero ganhar 3x o que gasto"        │
│ Target ROAS: 3.0                     │
│ Requer: Muitos dados históricos      │
│ Não recomendado: Fase inicial        │
└──────────────────────────────────────┘

MINha RECOMENDAÇÃO PARA VOCÊ:
Semana 1: "Maximizar Cliques"
Semana 2+: "Maximizar Conversões" com CPA = R$ 150

─────────────────────────────────────

CAMPO 3: Ajustes de Lance (Bid Adjustments)
────────────────────────────────────
Localização: Campaign Settings → Bid Adjustments

AUMENTAR LANCES (+):
├─ Mobile: +20%
│  Motivo: 70% das buscas são mobile, alta conversão
│
├─ Horário Comercial (8h-17h): +30%
│  Motivo: Pessoa está trabalhando, decisão
│
└─ Local (Centro de Goiânia): +15%
   Motivo: Mais perto do consultório, mais conversão

REDUZIR LANCES (-):
├─ Desktop: -10%
│  Motivo: Menos cliques mas não descartar
│
└─ Madrugada (22h-8h): -50%
   Motivo: Poucas buscas, conversão baixa

RESULTADO FINAL:
└─ Google paga MAIS pelas melhores oportunidades
└─ Paga MENOS pelas piores
└─ Otimização automática de orçamento

─────────────────────────────────────

CAMPO 4: Objetivos de Conversão
────────────────────────────────────
Localização: Conversions → Setup

PASSO 1: Definir Ações de Conversão

Conversão Primária:
✅ "Agendamento de Consulta"
   ├─ Ação: Envio de formulário
   ├─ URL: seu-site.com.br/agendamento/obrigado
   ├─ Valor: R$ 150 (custo médio de aquisição esperado)
   └─ Contagem: "Todos os eventos" (todo clique é valioso)

Conversão Secundária:
✅ "Formulário de Contato"
   ├─ URL: seu-site.com.br/contato/obrigado
   ├─ Valor: R$ 50 (lead, não consulta)
   └─ Contagem: "Todos"

✅ "Telefonema"
   ├─ Rastreamento: Código de chamada Google
   ├─ Valor: R$ 100
   └─ Duração mínima: 30 segundos

✅ "Email Coletado"
   ├─ URL: seu-site.com.br/email/confirmado
   ├─ Valor: R$ 20 (lead qualificado menor)
   └─ Contagem: "Todos"

PASSO 2: Implementação Técnica

Opção A: Google Analytics 4 (RECOMENDADO)
├─ Integração automática com Google Ads
├─ Rastreia eventos naturalmente
├─ Gratuito
└─ Conectar: Google Ads → Linked Google Analytics 4

Opção B: Google Ads Conversion Tracking
├─ Você coloca código no site
├─ Google rastreia automaticamente
├─ Funciona mesmo sem GA4
└─ Setup: Tools & Settings → Conversions → "+ Conversão"
```

```
CAMPO 2: Atribuição (Attribution Modeling)
────────────────────────────────────
Localização: Tools → Measurement → Attribution

O QUE É:
├─ Como Google "creditou" conversão
├─ Qual touchpoint (anúncio) recebeu crédito
└─ Importante para otimização

MODELOS DE ATRIBUIÇÃO:

Modelo 1: Last Click (Padrão)
├─ Crédito 100% para ÚLTIMO clique
├─ Exemplo: Clicou hipnose → clicou TEA → agendou
│           TEA recebe 100% do crédito
├─ Problema: Ignora hipnose que levou ele lá
└─ Uso: Campanha performance pura

Modelo 2: First Click
├─ Crédito 100% para PRIMEIRO clique
├─ Rastreia qual palavra-chave iniciou conversão
├─ Bom para: Entender início da jornada
└─ Uso: Otimizar topo do funil

Modelo 3: Linear
├─ Crédito igualmente distribuído
├─ Se 3 cliques: cada um recebe 33%
└─ Bom para: Visão holística

Modelo 4: Time Decay
├─ Crédito aumenta conforme perto da conversão
├─ Cliques recentes recebem mais crédito
└─ Bom para: Funil final (mais importante)

RECOMENDAÇÃO PARA VOCÊ:
├─ Fase 1: Last Click (entender conversão imediata)
├─ Fase 2: Linear (visão equilibrada)
└─ Fase 3: Time Decay (funil)
```

```
CAMPO 1: Pesquisa de Palavras-Chave
────────────────────────────────────
Ferramenta: Tools → Planning → Keyword Planner

PASSO 1: Pesquisar Termos
├─ Digitar: "hipnose ansiedade"
├─ Google retorna: Volume de busca, CPC, concorrência

PASSO 2: Analisar Resultados
┌──────────────────────────────────┐

├──────────────────────────────────┤
│ hipnose       | 12.100 | 2.50 | Alt│

│ hipnose ansie.| 2.400  | 3.20 | Med│
│ TEA adulto    | 1.900  | 1.80 | Med│
│ hipnose TEA   | 150    | 4.50 | Bx │
└──────────────────────────────────┘

INTERPRETAÇÃO:
├─ Volume alto = Muitas buscas
├─ CPC alto = Mais competição, custa mais
├─ Competição alta = Difícil ranking (genérico)
├─ Cauda longa (3+ palavras) = Menos volume, menor CPC
└─ Sua vantagem: Usar cauda longa especializada

─────────────────────────────────────

CAMPO 2: Adicionar Palavras-Chave ao Grupo
────────────────────────────────────
Local: Ad Groups → Keywords → "+ Keyword"

PASSO 1: Escrever Palavras-Chave
├─ 20-50 keywords por Ad Group
├─ Todas relacionadas ao mesmo tema
└─ Exemplo (Grupo "Hipnose-Ansiedade"):
   ├─ hipnose para ansiedade
   ├─ hipnose clínica ansiedade
   ├─ terapia hipnótica ansiedade
   ├─ hipnose ansiedade Goiânia
   ├─ consultório hipnose
   └─ ... (mais 15-20)

PASSO 2: Escolher Match Type
┌──────────────────────────────────┐
│ CORRESPONDÊNCIA EXATA [palavra]  │
├──────────────────────────────────┤
│ O que é:                         │
│ Aparece APENAS se busca exato    │
│                                  │
│ Exemplo:                         │
│ [hipnose para ansiedade]         │
│ ✓ "hipnose para ansiedade"      │
│ ✗ "hipnose ansiedade goiânia"   │
│                                  │
│ Vantagem: Controle total         │
│ Desvantagem: Menos volume        │
│                                  │
│ Uso: 40% de palavras-chave       │
└──────────────────────────────────┘

┌──────────────────────────────────┐
│ CORRESPONDÊNCIA DE FRASE "palavra"│
├──────────────────────────────────┤
│ O que é:                         │
│ Frase dentro da busca (qual ordem│
│ antes/depois não importa)        │
│                                  │
│ Exemplo:                         │
│ "hipnose para ansiedade"         │
│ ✓ "tratamento hipnose para ansied│
│ ✓ "hipnose para ansiedade online"│
│ ✗ "ansiedade tratamento hipnose" │
│                                  │
│ Vantagem: Bom equilíbrio         │
│ Desvantagem: Moderado            │
│                                  │
│ Uso: 35% de palavras-chave       │
└──────────────────────────────────┘

┌──────────────────────────────────┐
│ CORRESPONDÊNCIA AMPLA hipnose    │
├──────────────────────────────────┤
│ O que é:                         │
│ Qualquer ordem, qualquer contexto│
│                                  │
│ Exemplo:                         │
│ hipnose ansiedade               │
│ ✓ "ansiedade terapia hipnose"   │
│ ✓ "hipnose para fobia"          │
│ ✓ "psicólogo terapia"           │
│                                  │
│ Vantagem: Máximo volume          │
│ Desvantagem: Muito genérico      │
│                                  │
│ Uso: 25% de palavras-chave       │
└──────────────────────────────────┘

DISTRIBUIÇÃO RECOMENDADA:
├─ 40% Exata [palavra]
├─ 35% Frase "palavra"
└─ 25% Ampla palavra

─────────────────────────────────────

CAMPO 3: Palavras-Chave Negativas (Exclusão)
────────────────────────────────────
Local: Campaign/Ad Group → Keywords → Negative Keywords

O QUE SÃO:
├─ Palavras-Chave que você NÃO quer aparecer
├─ Economia de orçamento
└─ Evita cliques inúteis

EXEMPLOS PARA VOCÊ:

Se anuncia "Hipnose Clínica":
├─ -hipnose de palco (não clínico)
├─ -hipnose para emagrecer (outro segmento)
├─ -curso hipnose (você oferece serviço, não curso)
├─ -gratuito (você não oferece sessões grátis)
└─ -hipnose para parar fumar (não sua especialidade)

Se anuncia "TEA Diagnóstico":
├─ -autismo infantil (você atende adultos)
├─ -estimulação precoce (infantil)
├─ -terapia ABA (abordagem diferente)
├─ -teste autismo online (você quer offline)
└─ -aplicativo autismo (tecnologia, não serviço)

MATCH TYPES PARA NEGATIVAS:

Exata: [-palavra]
├─ Exclui APENAS palavra exata
├─ Mais restritivo
└─ Uso: 50%

Frase: -"palavra chave"
├─ Exclui frase e variações
├─ Bom equilíbrio
└─ Uso: 50%

AMPLAI: -palavra
├─ Exclui palavra em qualquer contexto
├─ Risco: Excluir demais
└─ Uso: Cuidado

QUANTIDADE:
├─ Mínimo: 20 negativas por campanha
├─ Recomendado: 50-100
└─ Atualizar: 1x por semana (revisar termos que chegam)
```
