### Para Anúncios

**O Que Funciona:**
```
✅ "Especialista em..." (credibilidade)
✅ "Primeira consulta grátis" (reduz barreira)
✅ "Resultado em X sessões" (expectativa clara)
✅ Números ("1000+ pacientes", "9+ anos")
✅ Nomes de métodos ("Ericksoniana", "AQ-28")
✅ Urgência suave ("Agenda limitada")
✅ Benefício tangível ("Dorme melhor", "Sem ansiedade")

❌ Evitar:
- Promessas absolutas ("Cura garantida")
- Comparações negativas ("Melhor que...")
- Jargão demais (use linguagem paciente)
- Muitos CTAs no mesmo anúncio
```

### Para Landing Pages

**Estrutura Psicológica:**
```
1. Hero (2 segundos): Entendo seu problema
2. Credibilidade (5 seg): Posso confiar?
3. Solução (30 seg): Funciona assim...
4. Prova Social (20 seg): Outros aprovam
5. CTA (Claro): O que fazer agora?
6. FAQ (Objeções): Respondidas
7. Conversão (Formulário): Simples
```

**Tempo de Leitura:**
- Hero: 5-10 segundos
- Descrição inteira: 2-3 minutos máximo
- Formulário: < 1 minuto preenchimento
