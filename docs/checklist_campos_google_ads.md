## 🎯 SEÇÃO 1: DADOS BÁSICOS DA CAMPANHA

### ✅ Nome da Campanha
```
ESTRUTURA: [TIPO]_[FOCO]_[ESTRATÉGIA]_[MÊS]

EXEMPLOS PARA VOCÊ:
├─ SEARCH_HipnoseAnsiedade_AltaIntencao_Jan2026
├─ SEARCH_TEADiagnostico_Informacional_Jan2026
├─ SEARCH_HipnoseTEA_Premium_Jan2026
└─ SEARCH_Remarketing_ClientesAntigos_Jan2026

BENEFÍCIO: Organizar múltiplas campanhas, fácil identificar em relatórios
```

### ✅ Tipo de Campanha
```
VOCÊ DEVE ESCOLHER: "Busca" (Search Network)

POR QUÊ?
├─ Maior intenção de compra (pessoa está buscando)
├─ Melhor para serviços (não produtos)
├─ Conversão mais alta que Display
├─ Menos disperdício de orçamento
└─ Perfecto para psicologia/terapia

NÃO USE:
├─ Display (banner ads - muito genérico)
├─ Shopping (para e-commerce)
├─ Video (YouTube - você não é creator)
└─ App (você não tem app)
```

### ✅ Objetivo da Campanha
```
ESCOLHER: "Conversões"

POR QUÊ?
└─ Você quer que cliquem E agendem (conversão)

ALTERNATIVAS:
├─ "Tráfego" (só cliques, sem tracking)
├─ "Consideração" (aumentar marca)
└─ "Vendas" (vender produtos - não é seu caso)

VOCÊ QUER: "Conversões"
```

## 💰 SEÇÃO 3: ORÇAMENTO & LANCES

### ✅ Orçamento Diário

```
CÁLCULO RECOMENDADO:

SE SEU OBJETIVO É: 5 consultas/mês

├─ Taxa conversão esperada: 15-20%
├─ Cliques necessários: 25-35 cliques
├─ CPC estimado: R$ 2,50-3,50
├─ Gasto necessário: 25 cliques × R$ 3,00 = R$ 75
├─ Com margem de segurança (20%): R$ 75 × 1,2 = R$ 90
└─ ORÇAMENTO DIÁRIO: R$ 100/dia

FÓRMULA GERAL:
(Cliques Desejados × CPC Estimado) ÷ 30 dias = Orçamento Diário

OPÇÕES:
├─ Teste (conservador): R$ 50-75/dia
├─ Crescimento: R$ 100-150/dia
├─ Scaling (se ROI positivo): R$ 200-300/dia
└─ Máximo (saiba o que está fazendo): R$ 500+/dia

MEU CONSELHO:
├─ Semana 1: R$ 50/dia (teste)
├─ Se funcionando bem, Semana 2: R$ 75/dia
├─ Se conversões, Mês 2: R$ 100-150/dia
└─ Mês 3+: Scale conforme retorno

CONFIGURAR NO GOOGLE ADS:
1. Campaign Settings
2. Budget
3. Type: Daily Budget
4. Daily Budget: R$ 100 (exemplo)
5. Save
```

### ✅ Método de Lances

```
ESCOLHA RECOMENDADA: "Maximizar Conversões"

POR QUÊ?
├─ Google usa IA para otimizar automaticamente
├─ Você não precisa mexer em lances manualmente
├─ Melhor resultado para iniciantes
└─ Google tenta atingir máximo de conversões no orçamento

FÓRMULAS AVANÇADAS (DEPOIS):
├─ Target CPA: R$ X por conversão
├─ Target ROAS: 3:1 (ganhar 3x o que gastou)
└─ Requer histórico de 20+ conversões

CONFIGURAR NO GOOGLE ADS:
1. Campaign Settings
2. Bidding
3. Choose: "Maximize Conversions"
4. Opcional: Set Max CPA to R$ 100 (máximo que quer pagar)
5. Save
```

### ✅ Ajustes de Lance (Bid Adjustments)

```
AUMENTAR LANCES (+):
├─ Mobile: +20% (mais buscas = mais oportunidade)
├─ Hora 9-17h: +30% (horário comercial, decisão)
└─ Local (centro de Goiânia): +15%

REDUZIR LANCES (-):
├─ Desktop: -10% (menos conversão que mobile)
├─ Madrugada (22h-8h): -50% (poucos buscam)
└─ Cidades distantes: -20%

RESULTADO:
└─ Google paga MAIS pelas oportunidades melhores
└─ Paga MENOS pelas piores
└─ Otimização automática de orçamento

CONFIGURAR NO GOOGLE ADS:
1. Ad Groups → Devices
2. Mobile: Click bid adjustment, type +20%
3. Do the same for Schedule (hora) and Location
4. Save
```

```
└─ "reprocessamento trauma hipnose"

GRUPO 3: TEA + Diagnóstico
├─ "diagnóstico autismo adulto"
├─ "avaliação TEA"
├─ "psicólogo autismo adulto"
├─ "teste autismo para adultos"
├─ "autismo feminino adulto"
├─ "síndrome asperger diagnóstico"
└─ "diagnosticar autismo tardio"

GRUPO 4: Hipnose + TEA (Premium)
├─ "hipnose TEA"
├─ "hipnose autismo adulto"
├─ "regulação emocional autista"
├─ "hipnose para autista"
└─ "fenômenos hipnóticos autismo"

TIPOS DE CORRESPONDÊNCIA (IMPORTANTE):

1️⃣ CORRESPONDÊNCIA EXATA [palavra-chave]
└─ Aparece APENAS se busca a expressão exata
└─ Exemplo: [hipnose para ansiedade]
└─ Busca: "hipnose para ansiedade" ✓ Aparece
└─ Busca: "hipnose ansiedade" ✗ NÃO aparece
└─ USO: Cauda longa, muito específico, controle total

2️⃣ CORRESPONDÊNCIA DE FRASE "palavra-chave"
└─ Aparece se a frase está no texto (qualquer ordem antes/depois)
└─ Exemplo: "hipnose ansiedade"
└─ Busca: "hipnose em Goiânia para minha ansiedade" ✓ Aparece
└─ Busca: "ansiedade tem cura hipnose" ✗ NÃO (ordem diferente)
└─ USO: Médio especificidade, bom equilíbrio

3️⃣ CORRESPONDÊNCIA AMPLA hipnose ansiedade
└─ Aparece se palavras aparecem em QUALQUER ordem, QUALQUER contexto
└─ Exemplo: hipnose ansiedade
└─ Busca: "ansiedade cura hipnose" ✓ Aparece
└─ Busca: "quero emagrecer com hipnose" ✓ Aparece (pode ser ruim!)
└─ USO: Volume alto, mais cliques, menos precisão

RECOMENDAÇÃO PARA VOCÊ:
├─ 40% Correspondência Exata [como-isto]
├─ 35% Correspondência de Frase "como isto"
└─ 25% Correspondência Ampla como isto

CONFIGURAR NO GOOGLE ADS:
1. Ad Groups
2. Keywords
3. Click "+ Keyword"
4. Add your keywords with correct match type
5. Save
```

### ✅ Palavras-Chave para EXCLUIR (Negative Keywords)

```
PROBLEMA: Você paga por cliques ruins

SOLUÇÃO: Palavras-chave NEGATIVAS (não mostrar anúncio)

EXEMPLOS PARA VOCÊ:

Se anuncia Hipnose Clínica:
├─ -hipnose de palco (entretenimento, não clínico)
├─ -hipnose para emagrecer (não sua especialidade)
├─ -hipnose para parar de fumar (pode ter, mas não seu foco)
├─ -curso de hipnose (você oferece serviço, não curso)
└─ -gratuito (você não oferece sessões grátis)

Se anuncia TEA Adulto:
├─ -autismo infantil (você atende adultos)
├─ -terapia ABA (abordagem diferente)
├─ -estímulo precoce (infantil)
├─ -vacinação autismo (você não discute isso)
└─ -pseudocura autismo (afasta pacientes sérios)

Se anuncia Hipnose + TEA:
├─ -como o Google rastreia
├─ -tecnologia autismo
├─ -apps para autismo (tecnologia, não terapia)
└─ -notícias autismo

RESULTADO:
└─ Cliques MAIS relevantes
└─ Taxa conversão SOBE
└─ CPA (custo por conversão) CAI

CONFIGURAR NO GOOGLE ADS:
1. Campaign (não ad group) → Keywords → Negative Keywords
2. Click "+ Negative Keyword"
3. Choose match type ([exata], "frase", ou ampla)
4. Add keywords to exclude
5. Save
```

## 📊 SEÇÃO 7: MONITORAMENTO & OTIMIZAÇÃO

### ✅ KPIs Essenciais (O Que Olhar)

```
DIARIAMENTE:

1️⃣ Impressões (Quantas vezes apareceu)
└─ Esperado: 50-200/dia (indica volume de busca)

2️⃣ Cliques (Quantas vezes clicaram)
└─ Esperado: 10-50/dia (depende orçamento)

3️⃣ CTR (Click-Through Rate)
└─ Fórmula: (Cliques ÷ Impressões) × 100
└─ Esperado: 5-10% é bom
└─ Se < 3%: Anúncio não é relevante, testar novo

4️⃣ CPC (Custo Por Clique)
└─ Esperado: R$ 2-4 para você
└─ Se > R$ 6: Palavra-chave muito cara, pausar

SEMANALMENTE:

5️⃣ Conversões (Consultas Agendadas)
└─ Esperado: 1-5 conversões/semana (para orçamento R$100/dia)

6️⃣ CPA (Custo Por Conversão)
└─ Fórmula: Gasto Total ÷ Número de Conversões
└─ Esperado: R$ 80-150 por consulta
└─ Se > R$ 200: Não lucrativo

7️⃣ Conversion Rate
└─ Fórmula: (Conversões ÷ Cliques) × 100
└─ Esperado: 15-25%
└─ Se < 10%: Landing page fraca

8️⃣ Quality Score (QS)
└─ Google dá nota: 1-10
└─ Esperado: 7-10
└─ Se < 5: Revisar palavra-chave, anúncio, landing page
└─ Afeta CPC diretamente (QS alto = CPC mais barato)

MENSALMENTE:

9️⃣ ROAS (Return on Ad Spend)
└─ Fórmula: Receita ÷ Gasto em Ads
└─ Esperado: 3:1 (ganhar 3x o gasto)
└─ Se < 2:1: Revisar estratégia

DASHBOARD RECOMENDADO:
└─ Criar planilha Google Sheets
└─ Copiar dados do Google Ads diariamente
└─ Gráficos: Tendência CPC, Conversões, CPA
└─ Revisar toda segunda-feira (reunião você mesmo)
```

### ✅ Otimizações (O Que Fazer)

```
SE CTR < 3%:
├─ Anúncio não é relevante
├─ Testar nova manchete
├─ Revisar se palavra-chave bate com anúncio
└─ Pausar anúncio após 1 semana se não melhorar

SE CPC > R$ 6:
├─ Palavra-chave muito competitiva
├─ Reduzir lance manualmente OU
├─ Pausar palavra-chave OU
├─ Usar cauda longa ("hipnose ansiedade Goiânia" em vez de "psicólogo")
└─ Focar em nichos menos competitivos

SE CONVERSÃO < 10%:
├─ Landing page fraca
├─ Testar nova landing page
├─ Verificar se CTA é claro
├─ Checar se formulário é muito longo
└─ Implementar WhatsApp flutuante para contato rápido

SE CPA > R$ 200:
├─ Não lucrativo (a menos que suas consultas custem > R$300)
├─ Pausar campanha
├─ Revisar publicidade (talvez público errado)
├─ Voltar ao drawing board
└─ Considerar contatar agência especializada

TESTES A/B (A Cada 2 Semanas):
├─ Teste 1: Manchete (urgência vs benefício)
├─ Teste 2: CTA ("Agendar" vs "Consulta Inicial Grátis")
├─ Teste 3: Oferta (com desconto vs sem)
└─ Deixar rodar 2 semanas, depois escolher melhor
```
